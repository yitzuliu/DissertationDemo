## 🔄 雙循環架構概述

### 第一循環：潛意識狀態感知（持續運行）
```
VLM觀察 → 視覺數字化 → State Tracker接收 → 比對引擎：RAG向量搜索 → 
決策：相似度判斷 → 更新白板狀態 → 存入滑動窗格 → 回到VLM觀察
```
- **特性**：用戶無感知背景運行，間隔2/3/5秒可調整，持續理解「我們現在進行到哪了」

### 第二循環：即時查詢響應（按需觸發）
```
用戶查詢 → State Tracker查看白板 → 直接讀取當前狀態 → 即時回應用戶
```
- **特性**：毫秒級響應（< 50ms），無需重新計算，直接從白板讀取完整狀態信息
