- **用戶查詢滿意度**：能正確回答「現在在哪步」、「下一步做什麼」等查詢

### 性能指標
- **響應時間**：用戶查詢響應 < 50ms，狀態更新 < 100ms，RAG搜索 < 10ms
- **記憶體使用**：系統總記憶體使用 < 50MB，滑動窗格 < 1MB

- **VLM容錯性**：VLM失敗率 < 30%時系統正常運行，連續失敗 > 10次時優雅降級
